## 💻 Getting started

**Clone the project and access the folder**

```bash
$ git https://github.com/toguvr/api_hubees && cd api-hubees
```

**Follow the steps below**

```bash
# install dependencies
$ yarn

# run docker-compose
$ docker-compose up -d

# run the migrations inside the container
$ yarn prisma migrate dev
```
